# 💧 Smart IV Fluid Monitoring System

## 📌 Project Overview
The Smart IV Fluid Monitoring System is a web application that monitors IV fluid levels and provides alerts when the fluid level becomes low. It helps reduce manual monitoring and improves patient safety in hospitals.

## 🚀 Features
- Real-time IV fluid level tracking
- Low-level alert notifications
- Simple and responsive dashboard
- User-friendly interface

## 🛠️ Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- shadcn UI
- Git & GitHub

## ▶️ How to Run
1. Clone the repository  
   `git clone https://github.com/SharanyaReddy27/smart-drip-monitoring.git`

2. Install dependencies  
   `npm install`

3. Start the project  
   `npm run dev`

4. Open in browser  
   `http://localhost:5173/`
